# ☁️ Data Redundancy Removal System

## 📌 Project Overview

The **Data Redundancy Removal System** is a Flask-based web application developed as **Task 1** for my Cloud Computing Internship. The system validates user data, identifies duplicate and false-positive records, and stores only valid records in a database, improving data accuracy and reducing redundancy.

---

## 🎯 Objectives

- Detect duplicate records before storing data.
- Validate email and phone number formats.
- Identify false-positive records.
- Store only unique and verified records.
- Improve database accuracy and efficiency.

---

## Features

Duplicate Email Detection
Duplicate Phone Detection
Data Validation
Delete Records
Responsive User Interface
